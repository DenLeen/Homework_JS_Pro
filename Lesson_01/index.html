ДЗ: Создать класс Sorter. У этого класса должен быть параметр - массив. У класса также должны быть методы: 1) add - добавление элементов в массив, возвращает длину массива; 2) length - возвращает длину массива; 3) sort - сортировка массива; 4) map - возвращает новый массив(по аналогии с стандартным методом); 5) filter - возвращает отфильтрованный новый массив(по аналогии с стандартным методом) 6) pushArray - по аналогии с методом push, только в pushArray нужно передавать другой массив, элементы которого должны быть добавлены
